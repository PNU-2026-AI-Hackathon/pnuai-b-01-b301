## 1. 자금 흐름 개요

```
[투자자]
  │
  │ 총 5,000,000원 청약
  │
  ▼
[에스크로 금고] ── 스마트컨트랙트가 보관
  │
  │ 마일스톤 달성할 때마다 운영자에게 공사비·운영비를 지급
  │ (투자자에게 돌려주는 게 아님 — 팜을 만드는 데 쓰는 돈)
  │
  ▼
[운영자] ── 지급받은 돈으로 팜 공사·장비 구매·운영
  │
  │ 팜에서 작물 재배 → 판매
  │
  ▼
[매출 발생] ── 작물 판매 수익
  │
  ├─ 운영비 차감 (임대료, 전기, 인건비 등)
  │
  ▼
[순이익]
  │
  ├─ 투자자 배당 60% (지분 비례)  ← 투자자가 돈을 버는 구간
  ├─ 운영자 보수 30%
  └─ 플랫폼 수수료 10%            ← 팜피가 돈을 버는 구간
```

## 3. 시나리오 정의

| 구분 | 🔴 보수적 | 🟡 기본 | 🟢 낙관적 |
|:---|:---|:---|:---|
| **재배 방식** | 단층 수경재배 | 다단식 3단 | 다단식 5단 |
| **작물** | 상추 단일 | 상추 + 바질 혼합 | 바질 + 마이크로그린 + 식용꽃 |
| **판매 채널** | 도매시장 출하 | 카페·식당 B2B 납품 | B2B 납품 + 로컬마켓 직판 혼합 |
| **kg당 단가** | 2,500원 (도매가) | 5,000원 (B2B 납품가) | 12,000원 (고부가 작물 직판) |
| **월 생산량** | 83kg | 200kg | 350kg |
| **전제 조건** | 판로 불안정, 최소 장비 | 카페 2~3곳 납품 계약 | 안정 납품처 + 주말 마르쉐 참여 |

> **근거:**
> - 단층 50㎡ 상추 생산량: 1,000㎡ 기준 연 20톤 → 50㎡ 비례 = 연 1,000kg ≒ 월 83kg
> - 다단식 3단: 재배면적 3배 → 월 ~200kg (로스 감안)
> - 다단식 5단: 재배면적 5배 → 월 ~350kg (고밀도 재배)
> - 상추 도매가 2,500원/kg, B2B 납품가 4,000~6,000원/kg, 바질·마이크로그린 직판 10,000~15,000원/kg

## 9. 시나리오별 핵심 판단

| | 🔴 보수적 | 🟡 기본 | 🟢 낙관적 |
|:---|:---|:---|:---|
| **결론** | 사업 불가 | 실현 가능 | 이상적 |
| **필수 조건** | — | 다단식 설비 + B2B 납품처 2~3곳 | + 고부가 작물 전환 + 직판 채널 |
| **최대 리스크** | 매출 < 임대료 | 납품처 이탈 시 매출 급감 | 고부가 작물 품질 관리 실패 |
| **현실 가능성** | 높음 (방치 시) | 중간 (노력 필요) | 낮음 (다수 조건 동시 충족) |
| **발표용 권장** | ❌ | ✅ 메인 시나리오 | 보조 참고용 |

**발표 전략:** 기본 시나리오를 중심으로 발표하되, 보수적 시나리오를 같이 보여주면서 "단층 도매로는 안 되고, 다단식 + B2B 납품이 최소 조건"이라는 점을 강조하면 현실 인식이 있다는 인상을 줄 수 있음.
